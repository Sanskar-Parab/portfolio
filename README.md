# Personal Portfolio

A sleek, responsive, and modern personal portfolio website built to showcase my software engineering skills, technical stack, and recent projects.

---

## 🚀 Features

*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
*   **Project Showcase:** Interactive displays of recent web development projects with source code and live links.
*   **Skill Grid:** Visual layout highlighting proficiency across modern languages, frameworks, and tools.
*   **Contact Integration:** Direct avenues for professional inquiries and social networking links.

---

## 🛠️ Tech Stack

| Category | Technologies Used |
| :--- | :--- |
| **Frontend** | React.js, Tailwind CSS, HTML5, CSS3 |
| **Build Tool** | Vite |
| **Deployment** | Vercel |
